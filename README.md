# VoiceFlow - Real-time EN ↔ FI Voice Translator

VoiceFlow is a premium, high-performance web application designed for seamless real-time speech translation between **English** and **Finnish**. 

## 🚀 Features

- **Live Voice Input**: Captures speech in real-time using the Web Speech API.
- **Instant Translation**: Translates speech as you finish your sentences.
- **Bi-directional Support**: Toggle effortlessly between English → Finnish and Finnish → English.
- **Premium UI**: Sleek dark-mode design with glassmorphism effects and smooth animations.
- **Visual Feedback**: Real-time "pulse" animation when the microphone is active.

## 🛠️ Technology Stack

- **HTML5/CSS3**: Semantic structure and modern glassmorphic styling.
- **JavaScript (ES6+)**: Core logic for speech recognition and API integration.
- **Web Speech API**: Handles high-accuracy speech-to-text directly in the browser.
- **MyMemory API**: A fast, client-side friendly translation engine.

## 📖 How to Use

1.  **Select Mode**: Use the toggle at the top right to choose your translation direction (English ↔ Finnish).
2.  **Start Listening**: Click the microphone button at the bottom.
3.  **Speak**: Start talking clearly. The "Original Speech" box will show your transcript.
4.  **See Results**: Once you stop speaking or pause, the translation will automatically appear in the box below.
5.  **Toggle**: Switch the language mode at any time to translate back in the other direction.

## ⚠️ Important Notes

- **Browser Compatibility**: This application uses the Web Speech API, which is best supported in **Google Chrome** and **Microsoft Edge**.
- **Permissions**: You must allow microphone access in your browser for the app to function.
- **Secure Context**: Web Speech API requires the application to be served over `HTTPS` or `localhost`.
